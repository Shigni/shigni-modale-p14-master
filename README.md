## How to install ?
```cmd
npm i shigni-modale-p14-master
```

## Parameters
Set the state
```javascript
const [displayModal, setDisplayModal] = useState(false);
```
Customize with :
```javascript
const modalParameter = {
	"backgroundColor": "#EEEEEE",   
	"borderRadius": 10,
	"boxShadow": "0 0 5px #1B1919",
	"color": "#1B1919",
	"fontSize": 18,
	"height": "fit-content",
	"padding": "20px 50px",
	"width": "fit-content"
}
```
